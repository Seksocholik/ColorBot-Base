# ColorBot Base in Rust

## What is added?
* Basic ScreenCapture
* Cool Obfuscation, Fake API Calls and Junk Code
* Lazy Imports (Static) and with it a Popups windows
* Hide Attribute in Lazy Static
  
  

## First Run
To build this, you will need:
* [Rust](https://rustup.rs/) (latest stable)
* [OpenCV](https://opencv.org/) installed on your system (and `clang` for the `opencv` crate)
* Also remember to put in enviroments of pc the opencv.
